# 🏋️ GymNaughy — תקציר בעברית

GymNaughy הוא פרויקט Full-Stack שמדגים בניית מוצר שלם מקצה לקצה סביב אפליקציית כושר:

- **`android-app/`** — אפליקציית אנדרואיד ב-Java (ארכיטקטורת MVVM) המיועדת למתאמנים: הרשמה והתחברות דרך Firebase, שלב onboarding לבחירת רמת כושר וציוד זמין, קבלת תוכנית אימונים מותאמת אישית, ביצוע אימון מודרך עם טיימר מנוחה, ומעקב רצף אימונים (streak) לשמירה על מוטיבציה.
- **`backend-api/`** — שרת API ב-Node.js/Express שמייצר את התוכנית המותאמת אישית, מתחבר ל-API חיצוני של תרגילי כושר (wger.de) עם שכבת מטמון, ומספק נקודות קצה למאמנים לצפייה בהתקדמות המתאמנים.
- **`web-dashboard/`** — דשבורד ווב ב-React למאמנים: רשימת מתאמנים עם אחוזי היענות ורצף אימונים, צפייה בהיסטוריית אימונים של מתאמן בודד, ועריכת תוכנית אימונים.

שלוש השכבות חולקות את אותו מודל נתונים (`User`, `Exercise`, `WorkoutPlan`, `WorkoutLog`) ומתקשרות דרך אותו REST API, עם התחברות משותפת דרך Firebase Authentication ואחסון ב-Firestore.

הפרויקט המקורי נבנה יחד עם שותף לצוות כאפליקציית Java/Android בלבד; הגרסה הזו היא הרחבה עצמאית שמוסיפה את שכבת ה-API ואת דשבורד ה-React, ומתעדת את הארכיטקטורה במלואה.

תיעוד מלא (ארכיטקטורה, מפרט API, מודל נתונים, הוראות הפעלה) נמצא בתיקיית [`docs/`](docs) ובקובץ [`README.md`](README.md) הראשי (אנגלית).
